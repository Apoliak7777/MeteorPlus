# Features
### Modules:
- Auto accept
- Auto craft plus (Autocraft + some additions)
- Auto leave
- Auto portal mine
- Auto repair
- Auto sell
- Bedrock storage brutedforce
- Boat aura
- Freeze
- Ghost block fixer
- Jesus Plus (Matrix bupass)
- Safe mine (Prevents player from lava)
- X-Ray bruteforce
